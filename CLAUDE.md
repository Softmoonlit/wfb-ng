# Claude Code Instructions

Please respond to all queries and conversations in Chinese. (一律使用中文回答)
All generated markdown documentation must also be in Chinese. (所有生成的markdown文档也必须使用中文)
All generated markdown document filenames must be in Chinese. (生成的markdown文档的文件名也必须使用中文)
All generated markdown document filenames should be concise and should not include a date prefix unless they are time-sensitive like meeting notes. (生成的markdown文档名应当简短，除非是会议记录等对时间敏感的文档，否则不需要加日期前缀)
All code comments must be in Chinese. (代码的注释也必须使用中文)
