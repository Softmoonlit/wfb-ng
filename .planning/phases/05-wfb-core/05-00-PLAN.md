# Plan 05-00: wfb_core 单进程入口重构

## Objective

重写 wfb_core.cpp，实现完整的单进程入口，支持 server/client 双模式，整合 Phase 1-4 实现的所有核心组件（Token 处理、调度器、FEC、TUN 设备）。

## Background

当前 wfb_core.cpp 仅包含测试骨架，所有函数直接返回。需要将其扩展为生产就绪的单进程入口，统一接管原有的 wfb_tx/wfb_rx/wfb_tun 功能。

### 5. 线程启动封装

```cpp
template<typename Func>
std::thread start_thread(const char* name, int priority, Func&& func) {
    std::thread t([name, priority, func]() {
        if (priority > 0) {
            set_thread_realtime_priority(priority);
        }
        func();
    });
    return t;
}
```
